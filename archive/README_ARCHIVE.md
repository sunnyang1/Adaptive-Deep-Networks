# Archive 说明

此目录包含重构过程中归档的旧代码。

## 目录结构

- `src_legacy/` - 旧 src/ 模块的备份（turboquant, rabitq/legacy 等）
- `QASP_legacy/` - QASP旧包备份
- `MATDO_legacy/` - MATDO旧实验代码
- `scripts_legacy/` - 旧脚本和wrapper
- `docs_legacy/` - 过时文档
- `experiments_legacy/` - 过时实验代码
- `old_contents/` - 重构前 archive/ 目录的原有内容

## 注意

归档的代码不再被活跃使用，仅供参考和历史回溯。
